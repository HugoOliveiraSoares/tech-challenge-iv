# Tech Challenge IV

Este projeto é uma solução para o Tech Challenge IV, com foco em uma arquitetura serverless para recebimento, classificação e processamento de avaliações de usuários.

A proposta contempla uma API de feedback, componentes de notificação para avaliações críticas e geração de relatórios periódicos, utilizando Java, Quarkus e serviços AWS conforme a especificação do desafio.

## Documentação

Os documentos de referência do projeto estão disponíveis em [`docs/`](docs/), incluindo a especificação técnica e o contrato OpenAPI da API de feedback.
